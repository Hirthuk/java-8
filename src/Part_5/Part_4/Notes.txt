Supplier Functional Interface

Represents a supplier of results.
There is no requirement that a new or distinct result be returned each time the supplier is invoked.

This is a functional interface whose functional method is get().

Use Cases:

Making the Database connection where it will just contains the logic of the database connection. No need to
return anything and others.

It won't take any Inputs as an argument
Supplier<Class> supplier = () -> logic
